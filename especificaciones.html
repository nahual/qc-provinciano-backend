Especificaciones Provinciano Backend

Backend para buscar informacion de las provincias argentinas.

API

  • GET https://sanadora.github.io/qc-provinciano-backend/index.html
    Query String
    • Parametro region: para filtrar por las provincias que pertenecen a esa region.
      Valores posibles:
      • Centro
      • Norte Grande Argentino
      • Nuevo Cuyo
      • Patagonica
      • Ninguna
    • Parametro nombre: para filtrar por provincias cuyo nombre coincide con lo especificado.
